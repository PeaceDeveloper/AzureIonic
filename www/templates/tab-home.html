Welcome to Ionic + Azure

This is the Ionic starter tabs app with a fully integrated Azure backend mobile service. To create a Backend on Azure, check out the Azure docs.

To edit the content of each tab, edit the corresponding template file in www/templates/. This template is www/templates/tab-dash.html

To learn about other mobile services offered by Azure such as Azure authentication, Push Notifications and Online/Offline sync, check outthis tutorial .
